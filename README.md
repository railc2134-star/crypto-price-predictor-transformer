# Crypto Price Predictor

A transformer-based model for predicting cryptocurrency prices.

## Overview
This project uses a transformer model to predict cryptocurrency prices based on historical data.

## Technologies Used
- PyTorch
- Python
- CSV

## How to Run
1. Install the required libraries by running `pip install torch` and `pip install pandas`.
2. Replace `BTC.csv` with your own dataset.
3. Run `python transfromer.py` to train the model or make predictions.
